{% comment %} _includes/navigation.html Site header + mega-menu nav. Menu items come from _data/navigation.yml — edit that file to add/remove/reorder links or dropdown children. On desktop, a top-level item with `children` is still a real link (clicking navigates) and the dropdown also opens via hover. On mobile, tapping a top-level item with `children` opens its dropdown instead of navigating (see main.js) — the dropdown's first entry is `all_label` (e.g. "All Dental"), a real link to the item's own `url`, since that's now the only way to reach the hub page on mobile. On mobile the whole menu opens as a full-screen panel below the logo/toggle bar, and the hamburger animates into an "X" (see main.js + _sass/_layout.scss). There is no "Home" nav item — the logo itself links back to "/". The "Free Consultation" button below is a good example of using the button include from within another include; change its text/url/style here if you want a different header CTA. {% endcomment %}
